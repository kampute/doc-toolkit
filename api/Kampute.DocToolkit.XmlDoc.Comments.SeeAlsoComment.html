Definition

Represents the documentation content for a see-also reference, which is associated with a code reference or hyperlink.
public class SeeAlsoComment : Comment
Inheritance
  • object

Remarks

The SeeAlsoComment class represents a <seealso> XML documentation tag, which is used to create cross-references between documentation topics. These can link to other code elements using a code reference (cref) or to external resources using hyperlinks (href).

Remarks

This method returns a new version of the see-also comment if its reference is a relative URL. The refinement process includes:
  • Applying the context's URL addressing strategy and translations to the reference
  • Using the target topic's title when the reference lacks descriptive text
Implementation Note
You typically do not need to call this method directly, as it is automatically invoked during the processing of XML documentation comments.
