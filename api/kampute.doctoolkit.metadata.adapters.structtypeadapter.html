Remarks

This class serves as a bridge between the reflection-based Type and the metadata representation defined by the IStructType interface. It provides access to struct-level information regardless of whether the assembly containing the struct type was loaded via Common Language Runtime (CLR) or Metadata Load Context (MLC).

Thread Safety

All public members of the type are guaranteed to be thread-safe.
