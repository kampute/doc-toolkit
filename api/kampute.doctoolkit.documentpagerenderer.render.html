Remarks

This method renders different types of documentation pages based on the specified category. The following table shows the mapping between page categories and model types:
Page CategoryModel Type
PageCategory.TopicTopicModel
PageCategory.NamespaceNamespaceModel
PageCategory.ClassClassModel
PageCategory.ClassWithMembersClassModel
PageCategory.StructStructModel
PageCategory.StructWithMembersStructModel
PageCategory.InterfaceInterfaceModel
PageCategory.InterfaceWithMembersInterfaceModel
PageCategory.EnumEnumModel
PageCategory.DelegateDelegateModel
PageCategory.ConstructorConstructorModel
PageCategory.ConstructorOverloadsOverloadCollection<T> where T is ConstructorModel
PageCategory.FieldFieldModel
PageCategory.EventEventModel
PageCategory.PropertyPropertyModel
PageCategory.PropertyOverloadsOverloadCollection<T> where T is PropertyModel
PageCategory.MethodMethodModel
PageCategory.MethodOverloadsOverloadCollection<T> where T is MethodModel
PageCategory.OperatorOperatorModel
PageCategory.OperatorOverloadsOverloadCollection<T> where T is OperatorModel
